    Come ottenere i file di configurazione dai provider di servizi WireGuard¶

    AzireVPN

    AzireVPN¶

    Sito ufficiale

    1. Accedi al sito ufficiale di AzireVPN ed effettua il login, poi apri il generatore di configurazione WireGuard.

    2. Nell'opzione IP, seleziona IPv4. Poi fai clic su Download Configuration.

    3. Poi continua seguendo questa guida.

    4. Puoi anche usare la app mobile per configurare AzireVPN.

    Hide.me VPN

    Hide.me VPN¶

    Sito ufficiale

    Hide.me VPN offre un modo semplice per usare il suo servizio WireGuard su un router GL.iNet.

    1. Collegati al router tramite SSH.

    2. Copia l'URL di installazione riportato sotto, incollalo nel terminale e premi Invio. Il clic destro del mouse incollera' il testo.

      curl -fsSL https://raw.githubusercontent.com/eventure/hide.client.routers/master/glinet_v4/hidemevpn | sh -s install

    3. L'installazione verra' avviata e poi verranno richiesti nome utente e password. Quando digiti o incolli la password, nel terminale non verra' mostrato alcun cambiamento; premi semplicemente Invio dopo averla digitata.

    4. Al termine, vai nel pannello di amministrazione web e vedrai che e' stato creato un gruppo hide.me VPN con i file di configurazione gia' presenti. Ti basta connetterti come faresti con qualsiasi altro file di configurazione.

    Nota: la chiave nel file di configurazione Hide.me VPN viene rigenerata prima di ogni connessione e diventa non valida dopo la disconnessione, quindi copiare questo file di configurazione su altri dispositivi non permettera' di connettersi correttamente.

    Mullvad

    Mullvad¶

    Sito ufficiale

    1. Accedi al sito ufficiale di Mullvad ed effettua il login, poi apri il generatore di file di configurazione WireGuard.

    2. Poi continua seguendo questa guida.

    3. Puoi anche usare la app mobile per configurare Mullvad.

    PIA (Private Internet Access)

    PIA (Private Internet Access)¶

    Sito ufficiale

    Non e' possibile scaricare le configurazioni WireGuard dal sito web; usa la app mobile per configurare PIA VPN.

    Surfshark

    Surfshark¶

    Sito ufficiale

    1. Se usi Surfshark, effettua il login e poi vai a questa pagina, fai clic su Router e seleziona WireGuard.

    2. Nella finestra successiva, seleziona I don't have a key pair.

    3. Seleziona Generate a new key pair.

    4. Una volta generata la chiave, seleziona Choose a location.

    5. Infine, scegli la posizione che vuoi configurare e premi il pulsante download accanto alla posizione. Potrai scaricare i file di configurazione.

    6. Poi continua seguendo questa guida.

    Link di riferimento

    AirVPN

    AirVPN¶

    Sito ufficiale

    1. Se usi AirVPN, accedi al sito, vai a Client Area e fai clic su Config Generator.

    2. Nella pagina Config Generator, seleziona WireGuard nella sezione Protocols.

    3. Seleziona un server, scorri fino in fondo e fai clic su Generate. Il file di configurazione verra' scaricato.

    4. Poi continua seguendo questa guida.

    Astrill

    Astrill¶

    Sito ufficiale

    Se usi Astrill, effettua il login e poi apri questa pagina per generare le configurazioni WireGuard.

    Poi continua seguendo questa guida.

    IVPN

    IVPN¶

    Sito ufficiale

    Se usi IVPN, devi generare manualmente la configurazione WireGuard. Segui la guida in base al tuo sistema operativo.

    Windows, macOS, Linux

    Poi continua seguendo questa guida.

    NVPN

    NVPN¶

    Sito ufficiale

    Segui la guida qui per creare la configurazione.

    Poi continua seguendo questa guida.

    OVPN

    OVPN¶

    Sito ufficiale

    1. Accedi a www.ovpn.com e trova il menu seguente per ottenere i file di configurazione WireGuard.

    2. Fai clic su Generate WireGuard keys, scegli il server desiderato e poi scarica la configurazione.

    3. Apri la configurazione con un editor di testo e copia il contenuto.

      La configurazione puo' contenere contenuti IPv6; poiche' i router GL.iNet non supportano IPv6 in modo sufficientemente completo, elimina il contenuto IPv6. Nell'esempio seguente, la parte evidenziata e' il contenuto IPv6.

    4. Poi continua seguendo questa guida.

    5. Puoi anche usare la app mobile per configurare OVPN.

    PrivateVPN

    PrivateVPN¶

    Sito ufficiale

    1. Effettua il login e poi apri il Control panel.

    2. Seleziona un server.

    3. Fai clic su GENERATE CONFIG, poi copia la configurazione.

    4. Poi continua seguendo questa guida.

    PrivadoVPN

    PrivadoVPN¶

    Sito ufficiale

    Accedi al sito di PrivadoVPN ed effettua il login.

    Nella dashboard, trova la sezione Manual Configuration e fai clic su WireGuard. Seleziona il server a cui vuoi connetterti, poi fai clic su Download Configration.

    Poi continua seguendo questa guida.

    Proton VPN

    Proton VPN¶

    Sito ufficiale

    Se usi Proton VPN, segui la guida qui per generare il file di configurazione WireGuard.

    Poi continua seguendo questa guida.

    PureVPN

    PureVPN¶

    Sito ufficiale

    Fai riferimento a questa guida oppure ai passaggi seguenti per ottenere manualmente il file di configurazione PureVPN.

    1. Accedi con il tuo account alla PureVPN Member Area. Dopo l'accesso, fai clic su Manual Configuration nel menu a sinistra.

    2. Seleziona la citta' di destinazione e fai clic su Download sulla destra.

    3. Nella finestra pop-up, seleziona WireGuard come protocollo.

    4. Seleziona Linux come tipo di dispositivo, quindi fai clic su Generate Configuration. Il file di configurazione verra' scaricato sul tuo dispositivo locale.

    Poi continua seguendo questa guida per completare la configurazione.

    Link di riferimento

    SpiderVPN

    SpiderVPN¶

    Sito ufficiale

    1. Accedi a www.spidervpn.org e trova la sezione per ottenere la configurazione VPN. Segui i passaggi per ottenerla.

    2. Scarica la configurazione VPN.

    3. Poi continua seguendo questa guida.

    StarVPN

    StarVPN¶

    Sito ufficiale

    1. Registra un account StarVPN

      Vai alle opzioni del piano tariffario e scegli il piano VPN piu' adatto alle tue esigenze. Puoi registrarti durante il checkout oppure direttamente qui.

    2. Scarica la configurazione WireGuard

      Accedi alla dashboard dell'area membri StarVPN. Fai clic su Wireguard Config per scaricare il file di configurazione. Ogni slot conterrà un file di configurazione WireGuard univoco.

      Suggerimento: se vuoi usare l'offuscamento AmneziaWG, fai clic su AmneziaWG Config per scaricare il file di configurazione. Ogni slot conterrà un file di configurazione AmneziaWG univoco.

    3. Modifica il file di configurazione, opzionale

      La configurazione puo' contenere un indirizzo IPv6. Per evitare problemi di compatibilita' e connettivita', apri il file .conf e rimuovi l'indirizzo IPv6, come mostrato di seguito.

    4. Poi continua seguendo questa guida per caricare il file di configurazione sul router GL.iNet.

    • AmneziaWG VPN Setup with StarVPN
    StrongVPN

    StrongVPN¶

    Sito ufficiale

    1. Se usi StrongVPN, accedi a https://wg.strongvpn.com.

    2. Seleziona una posizione dal menu a discesa, fai clic su GENERATE e apri il file di testo scaricato.

    3. Poi continua seguendo questa guida.

    4. Puoi anche usare la app mobile per configurare StrongVPN.

    TRUST.ZONE

    TRUST.ZONE¶

    Sito ufficiale

    1. Accedi a https://trust.zone/setup ed effettua il login.

    2. Scorri fino alla sezione WireGuard, scegli la porta che desideri e poi scarica una configurazione per un server specifico oppure un file zip con tutte le configurazioni.

    3. Poi continua seguendo questa guida.

    VPN.AC

    VPN.AC¶

    Sito ufficiale

    1. Se usi VPN.AC, devi accedere al pannello di controllo e trovare WireGuard Manager nel menu "Services".

    2. Crea la configurazione e scaricala.

    3. Poi continua seguendo questa guida.

    VPN Unlimited(KeepSolid)

    VPN Unlimited(KeepSolid)¶

    Sito ufficiale

    1. Se usi VPN Unlimited, accedi al tuo User Office > seleziona l'applicazione VPN Unlimited® > fai clic su Manage.

    2. Premi il campo sotto Device e fai clic su Manually create a new device… > imposta un nome personalizzato, ad esempio WireGuard > scegli la posizione appropriata del Server > seleziona il protocollo WireGuard® dal menu a discesa > fai clic su Generate.

    3. I parametri di configurazione appariranno quindi sotto in formato testo.

      Combina la configurazione come segue.

      [Interface]
      PrivateKey = incolla la PrivateKey dal tuo User Office
      ListenPort = incolla i dettagli di ListenPort
      Address = incolla le informazioni di Address
      DNS = incolla i dettagli DNS dal User Office

      [Peer]
      PublicKey = incolla la PublicKey dal tuo User Office
      PresharedKey = incolla i dettagli di PresharedKey
      AllowedIPs = incolla i dettagli di AllowedIPs
      Endpoint = incolla le informazioni di Endpoint

    4. Poi continua seguendo questa guida.
